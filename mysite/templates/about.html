A few words about us

A Web based application designed and developed using Django framework of Python programming language. Aim of the project is to make a web portal that can be used to guide / help people for getting knowledge about places before they visit it.

Some Exciting Features that you will see in this projects are:

  1. Viewers can see top 10 places in the screen.
  2. Viewers can get the reviews of the place from the comment section of the place.
  3. user can Search the place by its name or by searching the city name.
  4. User will get similar place recommendations too.
  5. User's feedback gets converted to ratings via sentiment analysis.
